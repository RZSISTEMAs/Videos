# 📦 Sistema de Airdrop Avançado (RZSISTEMA)

Este é um script Lua standalone para FiveM que adiciona um evento dinâmico de suprimentos ao seu servidor. Um avião cargueiro cruza o mapa e solta uma caixa com paraquedas protegida por mercenários.

## 🚀 Funcionalidades
- **Evento Cinematográfico:** Avião Titan cruza o céu e solta a carga no ponto designado.
- **Física de Paraquedas:** A caixa cai lentamente até o chão com física realista.
- **Sinalizador (Flare):** Ao pousar, a caixa solta fumaça vermelha para facilitar a localização.
- **Defesa NPC:** 3 Guardas armados da Merryweather protegem a carga contra saques.
- **Sistema de Saque:** Interação direta (tecla E) para abrir a caixa e receber recompensas.

## 🛠️ Instalação
1. Baixe a pasta `airdrop`.
2. Cole a pasta no diretório `resources` do seu servidor FiveM.
3. Adicione `ensure airdrop` ao seu arquivo `server.cfg`.
4. Reinicie o servidor ou dê um `refresh` e `start airdrop`.

## 🎮 Comandos
- `/airdrop` - Inicia o evento de queda de suprimentos próximo ao jogador.

# 📦 Advanced Airdrop System (RZSISTEMA)

This is a standalone Lua script for FiveM that adds a dynamic supply drop event to your server. A cargo plane crosses the map and drops a crate with a parachute, protected by mercenaries.

## 🚀 Features
- **Cinematic Event:** Titan plane crosses the sky and drops the cargo at the designated point.
- **Parachute Physics:** The crate falls slowly to the ground with realistic physics.
- **Flare Effect:** Upon landing, the crate releases red smoke to make it easier to find.
- **NPC Defense:** 3 armed Merryweather guards protect the cargo against looting.
- **Looting System:** Direct interaction (E key) to open the crate and receive rewards.

## 🛠️ Installation
1. Download the `airdrop` folder.
2. Paste the folder into your FiveM server's `resources` directory.
3. Add `ensure airdrop` to your `server.cfg` file.
4. Restart the server or run `refresh` and `start airdrop`.

## 🎮 Commands
- `/airdrop` - Starts the supply drop event near the player.
