# 🛡️ Painel P - Premium Admin Menu (RZSISTEMA)

A standalone administration system for FiveM with a modern, intuitive, and high-performance interface. Designed to simplify server management without heavy frameworks.

## 🚀 Features
- **Player Management:** Real-time list with Name/ID search and quick actions (Kick, Ban, Teleport, Kill).
- **Self Admin Tools:** Quick shortcuts for God Mode, Invisibility, Healing, and Vehicle Repair.
- **World Control:** Dynamic weather and time changes (Synchronized).
- **Permissions System:** Full control based on IDs (Initially configured for ID 1 as Owner).
- **Premium Interface:** Modern design with Glassmorphism, Vanilla CSS, and Vue.js for a fluid experience.

## 🛠️ Installation
1. Download the `painel_p` folder.
2. Paste into your `resources` directory.
3. Add `ensure painel_p` to your `server.cfg`.
4. Configure administrators in the `config.lua` file (Add desired IDs).
5. Restart the server.

## 🎮 Commands & Keys
- **[P] Key:** Opens/Closes the administrative menu (Only for those with permission).
