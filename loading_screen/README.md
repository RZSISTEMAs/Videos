# ⏳ Tela de Carregamento Personalizada (RZSISTEMA)

Uma tela de carregamento premium e moderna para FiveM, totalmente personalizada com vídeo de fundo local e logo pulsante.

## 🚀 Funcionalidades
- **Vídeo de Fundo Local:** Utiliza um arquivo `intro.mp4` para carregamento instantâneo e som garantido.
- **Logo Pulsante:** Nome do servidor "RZSISTEMA" com animação neon pulsante em CSS.
- **Barra de Progresso Real:** Mostra a porcentagem exata do carregamento do FiveM (Inicializando, Mapa, etc).
- **Som Automático:** Script para tentar desmutar o áudio automaticamente após o carregamento.

## 🛠️ Instalação
1. Baixe a pasta `loading_screen`.
2. Substitua o arquivo `intro.mp4` pelo seu vídeo de preferência (mantenha o nome `intro.mp4`).
3. Cole a pasta no diretório `resources` do seu servidor FiveM.
4. Adicione `ensure loading_screen` ao seu arquivo `server.cfg`.
5. Reinicie o servidor.

# ⏳ Custom Loading Screen (RZSISTEMA)

A premium and modern loading screen for FiveM, fully customized with local background video and pulsing logo.

## 🚀 Features
- **Local Background Video:** Uses an `intro.mp4` file for instant loading and guaranteed sound.
- **Pulsing Logo:** "RZSISTEMA" server name with CSS neon pulsing animation.
- **Real Progress Bar:** Shows the exact percentage of FiveM loading (Initializing, Map, etc).
- **Auto Audio:** Script to attempt to unmute audio automatically after loading.

## 🛠️ Installation
1. Download the `loading_screen` folder.
2. Replace the `intro.mp4` file with your preferred video (keep the name `intro.mp4`).
3. Paste the folder into your FiveM server's `resources` directory.
4. Add `ensure loading_screen` to your `server.cfg` file.
5. Restart the server.
